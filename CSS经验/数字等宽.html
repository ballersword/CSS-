# Helvetica Neue 字体 Helvetica Neue 字体 可以解决数字等宽问题 font-feature-settings: "tnum";：这个属性启用字体的 tabular numbers 特性，确保数字是等宽的。如果字体支持此特性，那么数字将按照等宽布局显示。 浏览器兼容性：有些浏览器可能需要带有前缀的属性来支持等宽数字功能，因此我们也提供了 -moz-font-feature-settings 和 -webkit-font-feature-settings 来兼容 Firefox 和 WebKit 内核的浏览器。 ~~~ *{font-family:Helvetica Neue;} ~~~
