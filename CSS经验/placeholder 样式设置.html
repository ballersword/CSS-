# placeholder 样式设置 input标签的placeholder，不支持样式设置，以下不是标准语法，不保证浏览器兼容。 ~~~ input::-webkit-input-placeholder{ color:#BCBEC0; } input::-moz-placeholder{ /* Mozilla Firefox 19+ */ color:#BCBEC0;}　　 input:-moz-placeholder{ /* Mozilla Firefox 4 to 18 */ color:#BCBEC0;} input:-ms-input-placeholder{ /* Internet Explorer 10-11 */ color:#BCBEC0;} ~~~
